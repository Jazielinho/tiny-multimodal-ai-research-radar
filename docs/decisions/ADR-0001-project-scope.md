# ADR-0001: Project Scope

## Status

Accepted.

## Context

The sprint has limited time and energy because it must coexist with work, CrossFit, family, heat and recovery.

## Decision

The project will focus on one main artifact:

**Tiny Multimodal AI Research Radar**

It will combine:

- LLM fundamentals
- RAG
- multimodal paper figures
- agent tools
- evaluation
- FastAPI/Docker demo

## Consequences

Positive:

- Less dispersion.
- Better portfolio coherence.
- Easier weekly tracking.
- Clearer professional positioning.

Negative:

- Kaggle, recommendation systems and YouTube automation are postponed.
- The first version will be small and imperfect.

## Review date

31 August 2026.
